# UI/UX 设计版本记录

## v1 - 2026-07-01

首个开发前 UI/UX 设计基线。

范围：

- 当前系统 UI 状态审计。
- Phase A-G 目标体验地图。
- 核心屏幕设计规格。
- 核心交互流程。
- 组件状态矩阵。
- Phase A-G 开发映射。

关键约定：

- assistant-ui 官方能力优先。
- 右侧面板只承载业务 Artifact。
- 设置、插件、审批、权限、审计进入完整设置页面。
- 主对话保留自然语言上下文、执行过程、审批摘要、Artifact 结果卡片。
- AI 动态 UI 采用受控 Artifact schema，不执行模型生成 JS。
