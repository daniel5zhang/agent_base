# 页面线框设计

版本：v1  
日期：2026-07-01  
用途：开发前页面级结构确认  
说明：本文档是低保真结构图，不定义最终视觉细节；最终实现仍优先使用 assistant-ui 官方组件和 shadcn/ui token。

## 1. App Shell 总览

目标：保持 assistant-ui 官方对话体验，把企业工作台能力分布到清晰区域。

```text
┌─────────────────────┬──────────────────────────────────────────────┬────────────────────────────┐
│ 左侧会话栏           │ 中间 Agent Thread                             │ 右侧业务 Artifact 面板       │
│                     │                                              │                            │
│ ┌─────────────────┐ │                                              │ ┌────────────────────────┐ │
│ │ Logo 产品名  [≡]│ │                                              │ │ Tab A  Tab B       [+] │ │
│ └─────────────────┘ │                                              │ ├────────────────────────┤ │
│ [+] 新建任务        │                                              │ │                        │ │
│                     │                                              │ │ Artifact Renderer       │ │
│ 今天                │                                              │ │                        │ │
│ • 会话 A            │                                              │ │                        │ │
│ • 会话 B            │                                              │ └────────────────────────┘ │
│ 昨天                │                                              │                            │
│ • 会话 C            │                                              │                            │
│                     │                                              │                            │
│ [设置与模型]        │                                              │                            │
└─────────────────────┴──────────────────────────────────────────────┴────────────────────────────┘
```

尺寸规则：

| 区域 | 默认 | 最小 | 最大 |
| --- | --- | --- | --- |
| 左侧会话栏 | 260px | 折叠 48px | 320px |
| 中间 Thread | 自适应 | 640px 可读宽度优先 | 约 42-48rem 内容宽 |
| 右侧业务面板 | 默认关闭 | 360px | 45vw |

## 2. 左侧会话栏

### 2.1 展开态

```text
┌────────────────────────────┐
│ ┌────┐ 企业 Agent 工作台 [≡] │
│ │logo│                      │
│ └────┘                      │
├────────────────────────────┤
│ + 新建任务                  │
│                            │
│ 今天                        │
│  帮我制定一阶段验收计划   … │
│  查 2026 惠民保总保费     … │
│  本地分析 10 20 30        … │
│                            │
│ 昨天                        │
│  用一句话介绍出师表       … │
│                            │
├────────────────────────────┤
│ ⚙ 设置与模型                │
└────────────────────────────┘
```

交互要求：

- 点击会话：仅加载该会话消息，不刷新整个会话列表。
- hover 会话：右侧显示 `…`。
- 当前会话：浅背景高亮。
- 点击 logo：在折叠态展开左侧栏。
- 点击折叠按钮：左侧滑出/滑入，动画与右侧一致。

### 2.2 折叠态

```text
┌────┐
│ ◎  │  logo，点击展开
│ +  │  新建任务
│    │
│    │
│ ⚙  │  设置与模型
└────┘
```

## 3. 中间 Agent Thread

### 3.1 空态

```text
                           今天要处理什么？

                 ┌────────────────────────────────┐
                 │ 给 Agent 发送消息               │
                 │                                │
                 │ 🔌  +                    ↑     │
                 └────────────────────────────────┘
```

规则：

- 不显示大段介绍。
- 不显示模拟任务列表。
- Composer 维持 assistant-ui Base 的圆角、阴影、留白节奏。

### 3.2 普通对话消息

```text
                                      ┌───────────┐
                                      │ 你好      │
                                      └───────────┘

        ╭─ 执行过程  v ─────────────────────────╮
        ╰───────────────────────────────────────╯

        你好！有什么我可以帮你的吗？

        [复制] [重新生成] [...]
```

规则：

- 执行过程默认折叠。
- 执行过程在本轮 assistant 回复之前。
- 回复正文流式输出。

### 3.3 工具调用消息

```text
                                      ┌──────────────────────────┐
                                      │ 查 2026 年惠民保总保费   │
                                      └──────────────────────────┘

        ╭─ 执行过程  v ─────────────────────────╮
        │ ✓ 识别意图：业务数据查询              │
        │ ✓ 检查权限：允许                      │
        │ ✓ 调用工具：问数插件                  │
        │ ✓ 生成结果：Artifact                  │
        ╰───────────────────────────────────────╯

        2026 年惠民保项目总保费为 12.36 亿元。

        ┌────────────────────────────────────────┐
        │ 2026 年惠民保项目总保费                │
        │ 总保费：12.36 亿元                     │
        │ 来源：问数插件 / 数仓                  │
        │ 审计编号：evt_xxx                      │
        │                              [打开结果] │
        └────────────────────────────────────────┘
```

## 4. 右侧业务 Artifact 面板

### 4.1 空态

```text
┌─────────────────────────────┐
│                             │
│                             │
│   从对话中的结果卡片        │
│   打开业务内容              │
│                             │
│                             │
└─────────────────────────────┘
```

规则：

- 不显示“业务面板”大标题。
- 不显示假 Tab。
- 不显示运行证据、诊断、审计编号等旧占位卡片。

### 4.2 有 Artifact Tab

```text
┌────────────────────────────────────────────┐
│ 惠民保总保费  x   明细表  x          [+]   │
├────────────────────────────────────────────┤
│                                            │
│ 标题：2026 年惠民保项目总保费              │
│                                            │
│ ┌────────────┐ ┌────────────┐              │
│ │ 总保费      │ │ 项目数      │              │
│ │ 12.36亿元   │ │ 86          │              │
│ └────────────┘ └────────────┘              │
│                                            │
│ [图表区域]                                  │
│                                            │
│ [表格区域]                                  │
│                                            │
│ [复制] [导出] [下载]                        │
└────────────────────────────────────────────┘
```

Tab 规则：

- hover Tab 显示关闭 `x`。
- 关闭 Tab 不删除后端 Artifact。
- 点击主对话卡片可重新打开。
- `+` 只打开已有可用 Artifact 或触发合法业务入口，不创建假面板。

## 7. 插件中心页

```text
设置与模型 / 插件

业务类型：[全部] [问数] [理赔] [投保] [查询]
状态：    [全部] [已启用] [未授权] [审批中] [不可用]

┌────────────────────────┐ ┌────────────────────────┐
│ 问数插件                │ │ 理赔查询插件             │
│ 内部业务插件             │ │ 内部业务插件             │
│ 状态：已启用             │ │ 状态：未授权             │
│ [使用] [停用] [详情]     │ │ [申请授权] [详情]        │
└────────────────────────┘ └────────────────────────┘

管理员管理 | 插件审计
```

规则：

- 普通用户不出现“安装”。
- 未授权插件只能申请授权。
- 已授权且启用后才出现在 Composer `+` 菜单。

## 9. 审计页

```text
设置与模型 / 审计

筛选：
[时间范围] [用户] [会话] [Run] [插件] [动作类型] [查询]

┌─────────────┬──────────┬──────────────┬────────────┬────────┐
│ 时间         │ 用户      │ 动作           │ 对象         │ 结果   │
├─────────────┼──────────┼──────────────┼────────────┼────────┤
│ 14:22       │ 张三      │ artifact.export │ 惠民保结果   │ allowed│
│ 14:20       │ 张三      │ api_key.view    │ Bailian     │ allowed│
└─────────────┴──────────┴──────────────┴────────────┴────────┘

[导出审计]
```

敏感字段：

- 默认脱敏。
- 查看明文需要权限和审计。

## 10. 移动端和窄屏约定

首阶段 Web 桌面优先，仍需预留响应规则：

| 屏幕 | 行为 |
| --- | --- |
| >= 1280px | 三栏布局 |
| 960-1279px | 左侧可折叠，右侧默认关闭 |
| < 960px | 左侧抽屉，右侧 Artifact 用全屏 sheet |
| 移动端 | Composer 固定底部，设置页全屏 |
