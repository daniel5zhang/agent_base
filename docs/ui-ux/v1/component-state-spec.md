# 组件状态矩阵

版本：v1  
日期：2026-07-01

## 1. ThreadListSidebar

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 初始加载 | skeleton 行 | 不阻塞主对话空态 |
| 空列表 | 只显示新建任务 | 不显示“暂无会话”等大块占位 |
| 已加载 | 按日期分组 | 点击切换 Thread |
| 当前选中 | 浅色背景 | 列表不重新加载 |
| hover | 显示更多按钮 | 可重命名/删除 |
| 删除当前会话 | 切到新任务空态 | 右侧面板关闭 |
| 折叠 | 只显示图标 | logo 点击展开 |
| 错误 | 小型错误提示 + 重试 | 不影响已加载会话继续使用 |

## 2. Composer

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 空输入 | 发送按钮 disabled | Enter 不发送 |
| 输入中 | 发送按钮 enabled | Enter 发送，Option+Enter 换行 |
| 发送中 | 停止按钮 | 可取消 Run |
| 附件选择 | 附件 chip | 可移除 |
| 插件选择 | 能力 chip 或本轮标记 | 仅影响本轮 Run |
| 服务不可用 | 错误提示 | 不清空用户输入 |
| 权限不足 | 引导申请授权 | 不假装执行成功 |

## 3. Assistant Message

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| reasoning | 折叠执行过程，低对比字体 | 可展开查看 |
| streaming | 正文打字机输出 | 自动滚动到底部 |
| tool_call | 工具调用过程 | 动态更新状态 |
| completed | 正文 + action bar | 复制/重试/更多 |
| failed | 错误摘要 + 重试 | 保留用户输入上下文 |
| blocked | 阻断原因 | 可进入申请或修改条件 |
| approval_required | 审批摘要卡 | 进入审批详情 |

## 4. Execution Process

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 默认 | 折叠行“执行过程” | 点击展开 |
| 运行中 | subtle 动态状态 | 不用大面积高亮 |
| 已完成 | 默认折叠 | 展开后显示 step 列表 |
| 失败 | 折叠行显示失败状态 | 展开看错误 |
| 取消 | 显示已取消 | 可重试 |

执行过程必须属于对应 assistant 消息，不能独立悬浮在顶部。

## 5. Artifact Link Card

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 可打开 | 标题、摘要、状态、来源 | 点击打开右侧 Tab |
| 加载中 | skeleton | 不阻塞主对话 |
| 权限不足 | 锁图标 + 申请入口 | 进入审批/授权 |
| 已失效 | 失效提示 | 可重新运行 |
| 需要脱敏 | 默认脱敏 | 明文查看需权限和审计 |

## 6. Right Artifact Panel

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 关闭 | 只显示展开按钮 | 点击右侧滑入 |
| 打开空态 | “从对话中的结果卡片打开业务内容” | 不显示假数据 |
| 有 Tab | Tab 行 + Renderer | 支持切换 |
| hover Tab | 显示关闭 x | 点击关闭 Tab |
| 最后 Tab 关闭 | 空态 | 面板可保持打开 |
| 加载 Artifact | skeleton | 不阻塞主对话 |
| Renderer 不支持 | JSON fallback | 提示 renderer 未注册 |
| 权限不足 | 权限提示 | 申请或返回 |

## 7. Settings Page

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 打开 | 完整页面 | 保留左侧设置菜单 |
| 加载 | 页面级 skeleton | 不显示旧数据误导 |
| 保存中 | 按钮 loading | 防重复提交 |
| 保存成功 | toast | 写审计 |
| 保存失败 | inline error | 保留表单输入 |
| 无权限 | 权限提示 | 可申请或返回 |
| 明文查看 | 短时展示 | 自动恢复脱敏并审计 |

## 8. Plugin Card

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 已授权未启用 | “启用” | 启用后进入 Composer + |
| 已启用 | “使用 / 停用” | 使用回到主对话 |
| 未授权 | “申请授权” | 进入申请流程 |
| 审批中 | “审批中” | 可查看申请 |
| 被管理员停用 | “不可用” | 普通用户不可启用 |
| 错误 | 错误原因 | 可重试加载 |

## 9. Approval Card

| 状态 | 展示 | 行为 |
| --- | --- | --- |
| 待提交 | 提交申请 | 生成申请 |
| 审批中 | 查看详情 | 进入设置页详情 |
| 已通过 | 继续执行 | 手动 resume |
| 已拒绝 | 重新提交 | 修改范围后提交 |
| 已过期 | 重新申请 | 新建申请 |
| 已复用 | 复用说明 | 写审计 |

## 10. Audit Sensitive Fields

| 场景 | 默认展示 | 明文展示 |
| --- | --- | --- |
| API Key | 脱敏 | 权限 + 二次确认 + 审计 |
| 数据结果敏感字段 | 脱敏 | 权限 + 审计 |
| 审批详情敏感范围 | 脱敏摘要 | 审批人/授权人可见 |
| 导出内容 | 不直接展示 | 权限 + 审批配置 + 审计 |
